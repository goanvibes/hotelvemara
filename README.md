# Hotel Vemara Goa Website

Static, mobile-first website redesign for Hotel Vemara, Panaji.

## Files
- `index.html` — homepage with booking enquiry panel
- `rooms.html` — room categories and amenities
- `dining.html` — restaurant and bar/lounge page
- `events.html` — banquet, meeting and conference page
- `location.html` — Google Maps embed and nearby distances
- `contact.html` — email and WhatsApp-style enquiry flow
- `assets/css/styles.css` — main styling
- `assets/js/main.js` — mobile menu, room filters, booking forms, FAQ and reveal animations
